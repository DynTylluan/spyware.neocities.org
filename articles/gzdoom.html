GZDoom

GZDoom is a source port of Doom based on an older source port, ZDoom.

Spyware Level: Low

GZDoom contains telemetry that it reports back to the developers containing general information about your Operating System, CPU Cores, and OpenGL compatibility.[1]. GZDoom's developers do not seem to be very good at handling privacy concerns (After all, if this was privacy-concious, it would be opt-in...) and so if you decide to use this program, you should make sure to compile it with the telemetry disabled, and you should make sure that this is the only spyware in the program- there may be more spyware implemented in the future.
